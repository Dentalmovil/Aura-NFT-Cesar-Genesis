# 💎 Aura NFT: Herencia de Campo
**By Dentalmovilr4**

Bienvenido al repositorio oficial de la colección **Génesis**. Este proyecto es una vitrina digital que captura la esencia de la ganadería y el paisaje del Cesar, Colombia, transformándola en activos digitales únicos (NFTs).

## 🚀 Visión
- **Red:** Desplegado inicialmente para visualización en OpenSea.
- **Ecosistema:** Explorando integración con **Monad** y redes de alta velocidad.
- **Tecnología:** HTML5, CSS3 (Hacker Aesthetic) y desplegado vía **Vercel**.

## 🖼️ La Colección
Nuestra serie cuenta con piezas que resaltan:
1. **Vínculo Ancestral:** La base de nuestra genética bovina.
2. **Energía Verde:** Procesos de nutrición y sostenibilidad.
3. **Horizonte de Control:** La majestuosidad del territorio.
